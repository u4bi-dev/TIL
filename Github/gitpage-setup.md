# 깃헙 페이지

## 깃헙 페이지 관하여
* 총 6가지의 단계를 거치면 정말 간단한 깃헙 페이지를 만들 수 있습니다.

* 깃헙 페이지 생성
    ```
    1. 레파지토리 생성
        respository : 깃헙이름.github.io

    2. 터미널/git bash 켜기
        git clone https://github.com/깃헙이름/깃헙이름.github.io

    3. cd 깃헙이름.github.io

    4. echo "hello world" > index.html

    5. 푸쉬하기
        git add .
        git commit -m 'init'
        git push -u origin master

    6. 완성!
    ```
